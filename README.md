# 📦 Lampada Library

A simple and efficient library for controlling lamps in embedded projects (such as Arduino, ESP32, Raspberry Pi, and similar platforms).

## 🚀 Features

* Digital lamp control (ON/OFF)

* PWM support (brightness control)

* Easy integration with existing projects

* Lightweight and optimized code

## 🛠 Requirements

* C++ (Arduino framework or similar)

* GPIO/PWM compatible board

## 📌 Included Examples

* ON/OFF – Basic lamp control

* Brightness – PWM brightness control

* Multiple Lamps – Control of multiple lamps
